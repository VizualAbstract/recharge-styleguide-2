Style Attributes

Attributes are added to the grid container element to give an additional layer of control to the grid system, such as reversing order or setting the max width

See examples below. To use them, add the HTML them to the <div class="rc_layout__container"> element, such as: <div class="rc_layout__container" rc_layout-order="reverse">

Reverse

Activated by using rc_layout-order="reverse". This will reverse the order of the columns

.rc_layout__sidebar

.rc_layout__content

<div class="rc_layout__container" rc_layout-order="reverse">
	<div class="rc_layout">
		<div class="rc_layout__sidebar">
			.rc_layout__sidebar
		</div>
		<div class="rc_layout__content">
			.rc_layout__content
		</div>
	</div>
</div>

Seamless

Activated by using rc_layout-style="seamless". This will remove the horizontal padding gaps

.rc_layout__sidebar

.rc_layout__content

<div class="rc_layout__container" rc_layout-style="seamless">
	<div class="rc_layout">
		<div class="rc_layout__sidebar">
			.rc_layout__sidebar
		</div>
		<div class="rc_layout__content">
			.rc_layout__content
		</div>
	</div>
</div>

Static

Activated by using .rc_layout-style="static". This will center the grid

rc_layout-style="static"

<div class="rc_layout__container" rc_layout-style="static">
	<div class="rc_layout">
		<div class="rc_layout__full">
			rc_layout-style="static"
		</div>
	</div>
</div>

Max-sizing

Activated by using rc_layout-max="lg". This will limit the max-width of the grid layout and center-align it. Works with xs, sm, md, lg

rc_layout-max="sm"

<div class="rc_layout__container" rc_layout-max="sm">
	<div class="rc_layout">
		<div class="rc_layout__full">
			rc_layout-max="sm"
		</div>
	</div>
</div>
