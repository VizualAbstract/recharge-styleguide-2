Radios

Radios allow the single-selection from an small array of options. Useful if multiple options need to be immediatly visible, unlike a dropdown select.

Using Radios

Label triggering

You can click on the label to activate the radio option, but the id and name attributes must be used in conjunction with eachother.

{{ ui_element("form/field_radio", {'label': 'Group 1', 'id': 'radio_group_1'}) | safe }} {{ ui_element("form/field_radio", {'label': 'Group 2', 'id': 'radio_group_2'}) | safe }}

Grouping radios

To make only one radio option selectable at any given time, they must all have the same name attribute.

{{ ui_element("form/field_radio", {'name': 'alignment', 'label': 'Left', 'id': 'alignment_left', 'checked': 'true'}) | safe }} {{ ui_element("form/field_radio", {'name': 'alignment', 'label': 'Center', 'id': 'alignment_center'}) | safe }} {{ ui_element("form/field_radio", {'name': 'alignment', 'label': 'Right', 'id': 'alignment_right'}) | safe }}

Radio groups

Radios are best visualized when bundled up within a .form__group element with several other options.
